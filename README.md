# MotionWiki

## Installation

### Required Tools

Install [npm](https://npmjs.org/), [bower](http://bower.io/), and [grunt](http://gruntjs.com/).

Then `cd` into the project directory:

    [prompt]$ cd path/to/motionwiki

### Install Libraries & Frameworks

    [prompt]$ npm install
    [prompt]$ grunt bowerful

### Build the project

    [prompt]$ grunt
